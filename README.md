# WebScraperCW2

COMP3011 Coursework 2 search engine tool for
https://quotes.toscrape.com/.

The project crawls the target website, builds an inverted index of the
words found on each page, stores the index as JSON, and provides a small
command-line shell for searching the saved index.

## Features

- Crawls all paginated quote pages from the target website.
- Respects the required 6-second politeness delay between requests.
- Builds a case-insensitive inverted index.
- Stores word frequency and word positions for each page.
- Saves and loads the compiled index from `data/index.json`.
- Supports single-word and multi-word search queries.
- Includes unit tests for crawler, indexer, search logic, and CLI handling.

## Setup

Create and activate a Python environment if desired, then install the
dependencies:

```bash
pip install -r requirements.txt
```

Dependencies:

- `requests` for HTTP requests
- `beautifulsoup4` for HTML parsing
- `pytest` and `pytest-cov` for testing

## Usage

Start the command-line shell from the repository root:

```bash
python -m src.main
```

The shell supports these commands:

```text
build
load
print <word>
find <query terms>
help
exit
```

### Build The Index

```text
> build
```

This crawls `https://quotes.toscrape.com/`, builds the inverted index, and
saves it to:

```text
data/index.json
```

Expected output is similar to:

```text
Crawled 10 pages.
Built index with 849 unique words.
Saved index to data\index.json.
```

### Load The Saved Index

```text
> load
```

Loads the compiled index from `data/index.json`.

### Print A Word's Inverted Index

```text
> print indifference
```

Example output:

```text
Inverted index for 'indifference':
- https://quotes.toscrape.com/page/2/ | title: Quotes to Scrape | frequency: 5 | positions: [449, 458, 467, 477, 483]
```

### Find Pages Matching A Query

```text
> find good friends
```

For multi-word queries, a page must contain every query word. Results are
ranked by the total word frequency across the query terms.

Example output:

```text
Found 2 page(s) for 'good friends':
- https://quotes.toscrape.com/page/2/ | title: Quotes to Scrape | score: 11
- https://quotes.toscrape.com/page/6/ | title: Quotes to Scrape | score: 3
```

## Testing

Run the test suite:

```bash
pytest
```

Run tests with coverage:

```bash
pytest --cov=src
```

The tests use fake HTML and mocked crawling behaviour where appropriate,
so they do not depend on the live website or wait for the real 6-second
politeness delay.


## Design Notes

The inverted index is stored as a nested dictionary:

```python
{
    "word": {
        "page_url": {
            "title": "Page title",
            "frequency": 2,
            "positions": [0, 5]
        }
    }
}
```

This structure makes `print <word>` efficient because each word maps
directly to all pages where it appears. Multi-word `find` queries are
implemented by intersecting the page URL sets for each query term.
